### Icons

```js
<MenuItem icon="Home">Home</MenuItem>
```

### Detail Text

```js
<MenuItem icon="Clock" detail="Get regular updates">
  Schedule
</MenuItem>
<MenuItem icon="Beaker" detail="Try the newest toys!">
  Labs
</MenuItem>
```

### Checkable

Can be used to display a menu showing the currently selected item.

```js
<MenuItem active>Gouda</MenuItem>
<MenuItem detail="Is often orange">Cheddar</MenuItem>
<MenuItem>Swiss</MenuItem>
```
